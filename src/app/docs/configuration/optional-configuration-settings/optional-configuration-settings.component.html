Optional Configuration Settings

There are various optional configuration options available for use. ##### `CacheStoreId` *(Optional)* Used to uniquely identify an application in the Cache Store so that cache keys between different application do not conflict. ##### `CacheStoreServiceUri` *(Optional)* Used to explicitly point to the Stateful Reliable Service that is the cache store. If not supplied the client will try to auto discover the cache store. ##### `CacheStoreEndpointName` *(Optional)* Used to explicitly specify the endpoint name that the Stateful Reliable Service cache store is listening on. If not supplied the client will use the default endpoint name for the cache store. ### How to Use Configuration Settings
